💡 "GitHub에 올리면 바로 실행" 정리
· 이 페이지(index.html)와 UI 프로토타입: GitHub Pages 설정 시 웹에서 바로 보임 ✓
· Flutter 앱(app/): 실제 모바일 앱이라 빌드 필요 (flutter run). GitHub에 올린다고 자동 실행되진 않음
· DB·콘텐츠: Supabase에 schema.sql + seed.sql 실행 후 사용

GitHub Pages 설정법 (UI 데모 공개)

  • 1. GitHub 저장소 → Settings → Pages
  • 2. Source: Deploy from a branch → Branch: main / (root)
  • 3. 저장 후 1~2분 → https://[사용자명].github.io/[저장소명]/ 접속
  • 4. 이 index.html이 자동으로 표시됨 → UI 프로토타입 버튼으로 데모 확인
