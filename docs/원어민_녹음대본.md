# 원어민 녹음 대본 — English Brain App

> 녹음 안내
>
> - 각 영어 문장을 **2번** 녹음: ① 보통 속도(natural) ② 천천히(slow, 또박또박)
> - 미국식 발음, 자연스러운 억양과 축약 그대로 (I've, don't, gonna 등)
> - 파일명 규칙: `s{번호}_natural.mp3` / `s{번호}_slow.mp3`
> - 한국어는 녹음 불필요 (앱에서 한국어 TTS 또는 텍스트로 처리)
> - 총 180개 문장
